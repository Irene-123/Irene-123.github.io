Quadratic Probing: Quadratic probing is an open-addressing scheme where we look for i2‘th slot in i’th iteration if the given hash vale x collides in the hash table.


How Quadratic Probing is done?

Let hash(x) be the slot index computed using the hash function.

  • If the slot hash(x) % n is full, then we try (hash(x) + 1*1) % n.
  • If (hash(x) + 1*1) % n is also full, then we try (hash(x) + 2*2) % n.
  • If (hash(x) + 2*2) % n is also full, then we try (hash(x) + 3*3) % n
  • This process is repeated for all the values of i until an empty slot is found.

Implentation in Python

def Quadratic_probing(array): n=len(array) table=[False]*(n) for i in range (0,n): hv=array[i]%n if table[hv]==False : table[hv]=array[i] else : for j in range (0,n): t=(hv+j*j)%n if table[t]==False : table[t]=array[i] break return table if __name__ == '__main__': x = int(input()) array = list(map(int, input().strip().split()))[:x] result=Quadratic_probing(array) print(result)
